## Aspire No Window Repro

### Setup
1. Download the [latest godot (.NET) binary](https://godotengine.org/download), and put it somewhere (e.g. Documents/Godot/Path)
2. Either
    * Rename e.g. Godot_v4.6.1-stable_mono_win64.exe to godot.exe, put folder on PATH

   OR
    * Update AppHost.cs with full path to executable

✅ Repo is on aspire 13.0.2, window appears  
❌ Update aspire to 13.1.0 or 13.2.0, fails to open window
